Welcome to Movie Catalog's Documentation!
======================================

**Movie Catalog** is a powerful desktop application for managing your personal movie collection.
It integrates with The Movie Database (TMDB) to automatically fetch movie metadata, posters, and details.

.. note::
    This documentation is for version 1.9.0 of Movie Catalog. For other versions, please check the `releases page <https://github.com/Nsfr750/movie_catalog/releases>`_.
